## 概要

COACHTECH 教材 Tutorial 7-2「制御構文 ハンズオン演習」で作成した成果物です。

生徒の成績データを元に、生徒ごとのテスト結果、テストの点数による判定をし、統計情報をまとめるwebアプリを作成しました

## 使用技術

- PHP 8.x
- html5

## 学んだこと

- 関数の定義方法（一つの関数には、一つだけのことをさせるべきであること）
- 条件分岐の使い方（elseにするべきか、elseifにするべきかの判断など）
- 二次元配列の定義、扱い方
- 繰り返し処理の使い方（配列を組み合わせたforeachの使い方など）

##　詰まったポイント

- foreachの使い方を勘違いしていて、二次元配列を用いてるのに、連想配列と同じ使い方をしてしまい、例えば、変数の中身が、値だと思っていたものの、実際には配列そのもので、理想とする表示がされなかった。
- echoでは、ダブルクウォートの中では配列のキーにはクウォートをつけないということと、配列のキーの記述とが、ごちゃごちゃになっていた
